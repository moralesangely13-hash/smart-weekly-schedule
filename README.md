# Smart Weekly Planner

Planner semanal fijo e interactivo creado con React + Vite + Tailwind CDN.

## Probar localmente

```bash
npm install
npm run dev
```

## Subir a GitHub

1. Crea un repositorio nuevo en GitHub.
2. Sube todos los archivos de esta carpeta.
3. Conéctalo a Vercel.

## Publicar en Vercel

- Framework: Vite
- Build command: `npm run build`
- Output directory: `dist`

## Archivo principal

`src/App.jsx`
